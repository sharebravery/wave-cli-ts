# 使用

安装：`npm install @sharebravery/wave-cli -g`

`wave -V`

# 初始化配置文件 init


`wave init`

# 规范化提交 commit

`wave commit`

`wave commit [msg]`

# 生成API
## OpenAPI To Typescript
`wave genapi:ts`
